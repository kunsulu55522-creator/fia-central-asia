# FIA image assets

Текущие изображения в этой папке считаются временными и подготовлены как структура для последующей замены на реальные материалы FIA.

## Current files

- `fia-logo.svg` — логотип/wordmark для шапки сайта. При наличии финального бренд-файла FIA замените этот файл, сохранив путь.
- `fia-hero-training.png` — временное Hero-изображение для главной, страницы команды, сертификации и Executive Coaching.
- `fia-programs-workshop.png` — временное изображение для страниц программ и основателя.
- `fia-central-asia-representation.png` — временное изображение для блока Казахстана и Центральной Азии.

## Recommended real FIA materials

- Official FIA logo: SVG, PNG, white version, favicon.
- Team photos: Наталия Вавилина, Натали ван Оттерло, Светлана Махат, Татьяна Рева.
- Founder photo: Берри Фаулер.
- Training photos: online training, group practice, coaching sessions, workshop moments.
- Certification visuals: FIA certificate sample and approved ICF-related visual materials.
- Hero images: official international education / coaching imagery in 16:9 or wider.

## Replacement rule

To avoid code changes later, keep the same filenames when replacing temporary assets.
