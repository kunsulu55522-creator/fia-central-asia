# FIA Russian Site

Полноценный проект сайта для Fowler International Academy of Professional Coaching — Русскоязычное отделение, с отдельным позиционированием генерального представительства FIA в Казахстане и Центральной Азии.

## Дизайн-система

**Визуальный тезис:** современная международная образовательная организация: светлая основа, строгая сетка, крупная типографика, реальные деловые визуалы и сдержанный премиальный акцент.

**Контентная логика:** Hero → Почему коучинг → доверие → региональное представительство → аудитории → программы → методология → ICF → команда → заявка.

**Интеракции:** плавное появление первого экрана, лёгкий подъём интерактивных блоков при наведении, устойчивые CTA без декоративной перегрузки.

### Цвета

- `ink` / графит: основной текст и тёмные секции.
- `navy` / `steel`: деловые акценты и основные CTA.
- `brass`: премиальный акцент для статусов и eyebrow-текстов.
- `moss`: WhatsApp и прямой контакт.
- `porcelain` / `mist`: светлые фоны страниц.

### Тональность

В ключевых разделах сайт говорит от первого лица: “мы обучаем”, “мы помогаем”, “наша академия”. Третье лицо используется для биографий, исторических фактов и объективной информации.

### ICF

На сайте используется согласованная формулировка: “Наш курс аккредитован Международной Федерацией Коучинга (ICF), что подтверждает его соответствие мировым стандартам качества”.

## Структура страниц

- `/` — Главная
- `/about` — О FIA
- `/programs` — Программы
- `/programs/basic` — Базовый курс профессионального коучинга
- `/programs/executive` — Executive Coaching
- `/programs/specializations` — Специализированные направления
- `/certification` — Сертификация и международные стандарты (ICF)
- `/founder` — Основатель
- `/team` — Наша команда
- `/contacts` — Контакты / заявка

## Запуск

```bash
npm install
npm run dev
```

Production-сборка:

```bash
npm run build
npm run start
```

## Vercel

Проект использует Next.js App Router и готов к публикации на Vercel. После подключения репозитория Vercel автоматически определит Next.js проект. Рекомендуемые настройки оставьте по умолчанию: build command `npm run build`, output managed by Next.js.
